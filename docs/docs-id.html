Dokumentasi Sabila

Panduan lengkap menggunakan Sabila untuk mempermudah workflow pengembangan web lokal Anda.

🇮🇩 Bahasa Indonesia 🇬🇧 English

1. Pengenalan

Sabila adalah lingkungan pengembangan web lokal pintar (*local web development environment*) yang dirancang khusus untuk membantu pemula, pelajar, dan developer membangun proyek web secara mudah, cepat, dan terotomatisasi menggunakan kecerdasan buatan (AI).

Sabila bertindak sebagai alternatif modern untuk XAMPP atau Laragon, namun dengan sentuhan asisten cerdas yang memanjakan developer.

Catatan: Sabila sangat direkomendasikan dipasang di direktori C:\sabila untuk menghindari konflik perizinan Windows Administrator.

2. Instalasi & Persiapan

Menginstal Sabila sangatlah mudah. Anda tidak perlu menyalin file secara manual, karena Sabila memiliki Setup Wizard cerdas.

  • Unduh Aplikasi: Kunjungi halaman Rilis GitHub dan unduh file .exe terbaru.
  • Jalankan Installer: Buka file hasil unduhan. Jika muncul peringatan Windows SmartScreen, klik More info > Run anyway.
  • Ikuti Setup Wizard: Pada saat pertama kali dibuka, Sabila akan meminta Anda memilih folder instalasi dan secara otomatis akan mengunduh komponen server (PHP, Nginx, MySQL, Node.js) dari internet.

3. Fitur Utama

Asisten Sabila AI 🤖

Di sudut kanan bawah aplikasi, Anda akan menemukan asisten AI. Anda bisa bertanya mengenai error kode, cara menghubungkan database, atau bahkan meminta AI menganalisis Log Server jika Apache/Nginx Anda tiba-tiba mati (crash).

Service Dashboard Terpadu 🎛️

Kontrol layanan web server terpopuler (Nginx, Apache, PHP, MySQL, phpMyAdmin) secara modular, responsif, dan rapi dalam satu panel.

Local Domain & SSL Dashboard 🔐

Buat virtual host (vhost) lokal dengan ekstensi domain khusus secara otomatis, lengkap dengan sertifikat SSL lokal sehingga website lokal Anda dapat diakses dengan aman (HTTPS).

Manajemen Multi-Version PHP & Node.js 🐘

Setiap proyek kadang butuh versi PHP atau Node yang berbeda. Di Sabila, Anda bisa membuka tab Alat Pengembang (Dev Tools) untuk mengganti versi aktif hanya dengan satu klik. Tidak ada lagi pusing mengatur Environment Variables (PATH) Windows secara manual.

Secret Manager (Security Shield) 🛡️

Simpan API Key penting (AWS, Stripe, dll) di penyimpanan persisten terenkripsi yang langsung disuntikkan ke dalam memori aplikasi dan terminal, tanpa membocorkannya ke file .env di disk.

Task & Cron Job Runner ⏳

Kelola dan jadwalkan tugas (background tasks) lengkap dengan fitur pemantauan status (Sukses/Gagal) dan baca output eksekusi langsung dari dashboard.

Auto-Tunneling Cloudflare 🌍

Ingin pamer website lokal Anda ke dosen atau klien? Buka pengaturan proyek, lalu klik Aktifkan Tunnel. Sabila akan menggenerasi URL publik aman (contoh: https://proyekmu.trycloudflare.com) yang bisa diakses dari seluruh dunia selama komputer Anda menyala.

Mail Catcher Lokal 💌

Bosan menggunakan Gmail asli untuk mengetes fitur Lupa Password? Sabila memiliki Mail Catcher terintegrasi. Semua email yang dikirim oleh aplikasi web Anda (di port SMTP 1025) akan langsung ditangkap dan bisa Anda baca melalui antarmuka Sabila. Tanpa bocor, 100% lokal.

Integrasi MCP (Model Context Protocol) 🔌

Integrasikan Sabila dengan agen AI favorit Anda (seperti Claude Desktop) agar AI dapat membaca repositori lokal dan log server Anda dengan aman.

4. Troubleshooting (Pertanyaan Umum)

Port 80 atau 3306 sudah dipakai aplikasi lain?

Jangan panik! Buka tab Sistem atau tanya ke Sabila AI. Sabila memiliki tombol Auto-Kill Port yang akan mematikan paksa aplikasi bandel (seperti Skype atau instalasi MySQL lawas) yang memblokir port tersebut dalam 1 detik.

Bagaimana cara mengubah Document Root?

Jika Anda terbiasa dengan XAMPP (menggunakan folder htdocs) atau Laragon (menggunakan www), Anda bisa merubahnya dari menu Pengaturan. Sabila akan otomatis memindahkan file dan mengatur ulang konfigurasi Nginx/Apache untuk Anda.
