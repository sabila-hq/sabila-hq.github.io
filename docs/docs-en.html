Sabila Documentation

The complete guide to using Sabila to simplify your local web development workflow.

🇮🇩 Bahasa Indonesia 🇬🇧 English

1. Introduction

Sabila is a smart local web development environment specifically designed to help beginners, students, and developers build web projects easily, quickly, and automatically using Artificial Intelligence (AI).

Think of Sabila as a modern alternative to XAMPP or Laragon, but supercharged with an intelligent assistant tailored for developers.

Note: It is highly recommended to install Sabila in the C:\sabila directory to avoid Windows Administrator permission conflicts.

2. Installation & Setup

Installing Sabila is extremely easy. You don't need to copy files manually; Sabila comes with an intelligent Setup Wizard.

  • Download the App: Visit the GitHub Releases page and download the latest .exe file.
  • Run the Installer: Open the downloaded file. If Windows SmartScreen appears, click More info > Run anyway.
  • Follow the Setup Wizard: On the first launch, Sabila will ask you to select an installation folder and will automatically download the necessary server components (PHP, Nginx, MySQL, Node.js) from the internet.

3. Key Features

Sabila AI Assistant 🤖

In the bottom-right corner of the application, you'll find the AI assistant. You can ask about code errors, how to connect to databases, or even ask the AI to analyze your Server Logs if Apache/Nginx crashes unexpectedly.

Unified Service Dashboard 🎛️

Control popular web server services (Nginx, Apache, PHP, MySQL, phpMyAdmin) in a modular, responsive, and neat single panel.

Local Domain & SSL Dashboard 🔐

Automatically create local virtual hosts (vhosts) with custom domain extensions, complete with local SSL certificates so your local website can be accessed securely via HTTPS.

Multi-Version PHP & Node.js Management 🐘

Different projects sometimes require different PHP or Node versions. In Sabila, you can open the Developer Tools tab to switch active versions with just one click. Say goodbye to the headache of manually editing Windows Environment Variables (PATH).

Secret Manager (Security Shield) 🛡️

Store important API Keys (AWS, Stripe, etc.) in encrypted persistent storage that is injected directly into application and terminal memory, without leaking them to .env files on disk.

Task & Cron Job Runner ⏳

Manage and schedule background tasks complete with execution status monitoring (Success/Failed) and read execution output directly from the dashboard.

Cloudflare Auto-Tunneling 🌍

Want to show off your local website to a professor or client? Open your project settings, then click Enable Tunnel. Sabila will generate a secure public URL (e.g., https://yourproject.trycloudflare.com) accessible worldwide as long as your computer is on.

Local Mail Catcher 💌

Tired of using your real Gmail account to test "Forgot Password" features? Sabila has an integrated Mail Catcher. All emails sent by your web app (via SMTP port 1025) will be caught immediately and can be read directly in the Sabila interface. 100% local, no leaks.

MCP (Model Context Protocol) Integration 🔌

Integrate Sabila with your favorite AI agents (like Claude Desktop) so the AI can securely read your local repository and server logs.

4. Troubleshooting (FAQ)

Port 80 or 3306 is already in use?

Don't panic! Open the System tab or ask Sabila AI. Sabila has an Auto-Kill Port button that force-stops rogue applications (like Skype or old MySQL installations) blocking those ports in just 1 second.

How do I change the Document Root?

If you're used to XAMPP (which uses the htdocs folder) or Laragon (which uses www), you can switch it directly from the Settings menu. Sabila will automatically move your files and reconfigure Nginx/Apache for you.
